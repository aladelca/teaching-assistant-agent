# Code of Conduct

## Our Pledge
We are committed to making participation in this project a respectful and
harassment-free experience for everyone.

## Expected Behavior
- Be respectful and constructive in discussions and reviews.
- Focus feedback on code, design, and evidence.
- Assume good intent and ask clarifying questions before escalating.
- Protect privacy and avoid sharing student-sensitive data.

## Unacceptable Behavior
- Harassment, insults, intimidation, or discrimination.
- Personal attacks or demeaning language.
- Publishing private information without explicit permission.

## Scope
This Code of Conduct applies in project spaces, including issues, pull requests,
code review threads, and other communication channels managed by maintainers.

## Reporting
Report incidents privately to project maintainers.
Include context, timestamps, and any relevant links or screenshots.

## Enforcement
Maintainers may remove comments, reject contributions, or restrict participation
for behavior that violates this Code of Conduct.
